Embedding Media (Starter)

This page is intentionally plain. Your job is to edit the HTML tags and attributes to see how embedding works.

1) Images: <img>

2) Audio: <audio>

3) Video: <video>

4) iFrame: <iframe>

5) CSP Basics (Concept Only)

If an embed fails, open DevTools Console and look for an error message.

If you see errors like “Refused to load ... because of Content Security Policy”, it usually means the page’s CSP did not allow that domain.

Practice checklist

  1. Change the image src and alt.
  2. Add loop to the audio element.
  3. Add a poster to the video element.
  4. Embed a different site/video using an iframe.

Reminder: This starter file is HTML-only on purpose.
