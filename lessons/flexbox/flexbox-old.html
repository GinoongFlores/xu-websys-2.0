CSS Flexbox Basics

Learn how Flexbox arranges items in a row or column and makes spacing easy.

What Is Flexbox?

Flexbox is a CSS layout system for arranging items in a single direction: row or column. It is great for navigation bars, card layouts, and aligning content inside containers.

Flexbox always starts with a container (the parent) and items (the children). You turn a container into a flex container using display: flex.

Main Axis vs Cross Axis

Flexbox aligns items using two invisible lines called axes. The main axis follows the direction of flex-direction. The cross axis runs perpendicular to it.

A quick rule: justify-content works on the main axis, and align-items works on the cross axis.

Remember

Change flex-direction, and the main axis flips. The axis names never change, but their direction does.

Flex Container Basics

The container controls the main direction, spacing, and alignment. Start with these core properties:

  • display: flex turns on Flexbox.
  • flex-direction sets row or column.
  • justify-content aligns along the main axis.
  • align-items aligns along the cross axis.
  • gap adds space between items.
CSS Example .card-row {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 12px;
}

Wrapping and Flexible Sizes

When you have many items, you can allow wrapping with flex-wrap. Items can also grow or shrink using the flex shorthand.

CSS Example .tag-list {
  display: flex;
  flex-wrap: wrap;
  gap: 10px;
}

.tag {
  flex: 1 1 140px;
}

Aligning Individual Items

You can override alignment for a single item with align-self. This is useful when one card or button needs special positioning.

CSS Example .profile-row {
  display: flex;
  align-items: flex-end;
}

.profile-row .highlight {
  align-self: center;
}

Quick Flexbox Checklist

  • Start with a parent container and use display: flex.
  • Use flex-direction to pick row or column.
  • Align items with justify-content and align-items.
  • Allow wrapping with flex-wrap for long rows.
  • Fine-tune individual items with flex and align-self.
