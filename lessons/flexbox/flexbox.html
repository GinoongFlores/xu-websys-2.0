What Is Flexbox?

Flexbox is a CSS layout system for arranging items in a row or column. It works best when you have a parent container and several children.

Your task: create a parent container, give it display: flex, and then experiment with alignment.

Key Concept: Flexbox always has a parent (the flex container) and children (the flex items). The parent controls how the children are arranged!

Main Axis vs Cross Axis

Flexbox uses two axes. The main axis follows flex-direction, and the cross axis runs perpendicular to it.

Remember: justify-content aligns on the main axis and align-items aligns on the cross axis.

Flex Container Basics

The container controls the main direction, spacing, and alignment. Start with these core properties:

  • display: flex turns on Flexbox.
  • flex-direction sets row or column.
  • justify-content aligns along the main axis.
  • align-items aligns along the cross axis.
  • gap adds space between items.

Wrapping and Flexible Sizes

Create a row of tags that wraps onto a new line when there is not enough space. Use flex-wrap: wrap to enable wrapping!

Interactive Demo: Wrapping Tags (Try Resizing Your Browser!)
HTML CSS Flexbox Layout Responsive Design Web Dev
Try This CSS: 
.tag-list {
  display: flex;
  flex-wrap: wrap;
  gap: 12px;
}

.tag {
  padding: 10px 20px;
  border-radius: 20px;
}

Tip: Without flex-wrap, items will shrink to fit in one line. With wrapping, they move to a new line when space runs out!

Aligning Individual Items

Use align-self to move one item while the rest stay aligned together. This overrides the container's align-items setting!

Try This CSS: 
.profile-row {
  display: flex;
  align-items: flex-end;  /* Default for all items */
  gap: 15px;
}

.highlight {
  align-self: center;  /* Override for this one item */
}

Notice: The first and third items align to the bottom (flex-end), but the middle one uses align-self: center to position itself differently!

Flexbox Checklist

Follow this checklist to master Flexbox basics:

  • Apply display: flex to the parent container.
  • Choose row or column with flex-direction.
  • Use justify-content to align on the main axis.
  • Use align-items to align on the cross axis.
  • Try wrapping with flex-wrap if items overflow.
  • Use align-self for one-off adjustments on individual items.
  • Add gap to create space between flex items easily!

Practice Tip: Open your browser's Developer Tools (F12) and experiment with these properties in real-time. Change values and see what happens!
