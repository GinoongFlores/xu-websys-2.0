    2) HTML Entities

    Rendered output

    We can show angle brackets: <div>

    We can show ampersands: Tom & Jerry

    Copyright symbol: © (also: ©)

    Entities make sure your text displays correctly.

    What the HTML looks like

    <p>We can show angle brackets: &lt;div&gt;</p>
<p>We can show ampersands: Tom &amp; Jerry</p>
<p>Copyright: &copy; (also: &#169;)</p>

    3) Whitespaces

    Collapsed whitespace (default)

    Hello world! This line break will become a space. And these extra spaces will collapse.

    Look closely: you’ll see normal spacing only.

    Preserved whitespace (CSS: white-space)

    Hello world! This line break stays a line break. And these extra spaces stay.

    This is useful for showing code, lyrics, or formatting.
