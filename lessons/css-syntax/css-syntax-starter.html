Understanding CSS Syntax

Learn about declarations, blocks, rulesets, and statements

1. CSS Declarations

A declaration is a property-value pair separated by a colon. Every declaration must end with a semicolon.

Syntax:

property: value;

Examples:

color: blue;
font-size: 16px;
margin: 10px;
background: #fff;

2. CSS Declaration Blocks

A declaration block is a group of declarations wrapped in curly braces {}. Multiple declarations are separated by semicolons.

Syntax:

{
  property1: value1;
  property2: value2;
  property3: value3;
}

Example:

{
  color: white;
  background: navy;
  padding: 20px;
}

3. CSS Rulesets

A ruleset combines a selector with a declaration block. It tells the browser which elements to style and how to style them.

Syntax:

selector {
  property: value;
  property: value;
}

Examples:

/* Style paragraphs */
p {
  color: #333;
  line-height: 1.6;
}

/* Style a class */
.button {
  background: green;
  color: white;
  padding: 10px;
}

/* Style an ID */
#header {
  background: black;
  color: white;
}

4. CSS Statements

A statement is a complete CSS instruction. There are two types:

Type 1: Rulesets (Regular Rules)

Selector + declaration block:

/* A simple ruleset statement */
h1 {
  color: #667eea;
  font-size: 2rem;
}

Type 2: At-Rules (@-rules)

Special statements that start with @:

Media Query (responsive design):

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}

Import (load external styles):

@import url('styles.css');

5. Quick Reference

Term What It Is Example
Declaration Property-value pair color: blue;
Declaration Block Group of declarations in { } { color: blue; margin: 10px; }
Ruleset Selector + declaration block p { color: blue; }
Statement Complete CSS instruction p { ... } or @media ...
